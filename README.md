# SDS.Avalonia

Windows でサウンドデバイスを切り替えるアプリです。

Windows11 で、マルチスクリーンでゲーム中に、サブディスプレイでデバイスの切り替えが不便だったので、作成しました。

AvaloniaUI 12 で NativeAOT 対応で書き直しました。

## 使い方

起動すると、使用できるデバイスが全て列挙されます。

デバイス名の付近がくそでかボタンになっているので、それをクリックすることでデバイスを切り替えられます。

押せなくなっているのが現在のデバイスです。

通常多数のデバイスがインストールされているはずなので、各デバイスの下部に「Bookmark」のチェックボックスを用意しています。

普段使うデバイスをブックマークして、右上の「ブックマークのみ」で必要なもののみを表示して使うことを想定しています。

上部のスライダで音量の調整も出来ます。その横の Mute でミュートも出来ます。
